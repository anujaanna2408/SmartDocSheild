# SmartDocShield | Secure Document Intelligence

SmartDocShield is a futuristic AI-powered system for automated text extraction, PII (Personally Identifiable Information) detection, and data masking from sensitive documents.

## 🚀 Deployment

The project is configured for deployment on **Render**.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/anujaanna2408/SmartDocSheild)

## ✨ Features

- **Document Processing**: Support for PDF, DOCX, TXT, and various image formats (PNG, JPG, etc.).
- **OCR Engine**: Advanced optical character recognition powered by EasyOCR and Tesseract.
- **NLP Intelligence**: Named Entity Recognition (NER) using SpaCy to identify key entities and phrases.
- **PII Protection**: Automated detection and masking of sensitive information (PAN, Aadhaar, SSN, Emails, etc.).
- **Summarization**: Extractive AI-driven document summarization.
- **Search & Archive**: Comprehensive search system for managing and retrieving processed history.

## 🛠️ Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6+), Futuristic Glassmorphic UI.
- **Backend**: Python, Flask, Gunicorn.
- **AI/ML**: SpaCy, NLTK, EasyOCR, Tesseract, PyPDF2.
- **Database**: SQLite3.

## 📦 Local Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/anujaanna2408/SmartDocSheild.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the development server:
   ```bash
   python server.py
   ```
4. Access the dashboard:
   `http://localhost:5000`
